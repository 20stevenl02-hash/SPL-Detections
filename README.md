# 🔍 SPL Detection Library

A curated collection of high-quality **Splunk Processing Language (SPL)** detections designed to identify, investigate, and respond to real-world security threats.

---

## 📌 Overview

This repository showcases custom-built SPL detections developed to enhance visibility across enterprise environments. The focus is on:

* Detecting adversary behavior aligned with known attack techniques
* Reducing false positives through practical tuning
* Supporting threat hunting and incident response workflows
* Providing clear, reusable detection logic

Whether you're a SOC analyst, detection engineer, or security researcher, these queries are built to be **actionable and adaptable**.

---

## 🧠 Detection Philosophy

Each detection in this repository is created with the following principles:

* **Behavior over signatures** – Focus on attacker techniques, not just indicators
* **Context-aware** – Leverages enrichment and environmental baselines
* **Explainable logic** – Queries are written for readability and transparency
* **Operational value** – Designed for real-world SOC use, not just theory

---

## ⚙️ Usage

1. Copy the SPL query into your Splunk environment
2. Adjust index names and field mappings as needed
3. Apply environment-specific tuning
4. Convert into alerts, dashboards, or hunting queries
